- Evaluated deployment options for Wingman — Flutter app is retired/abandoned in favour of a web UI approach

- **Drop Cursor-specific files** — all tools point at the same cache files
- **Cache files renamed** to `ccontext.md` and `cprompt.md` (avoids terminology clashes)
- **Migration support** required for existing projects using the old wingman structure
